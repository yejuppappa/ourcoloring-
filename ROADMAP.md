### Grok API 모델 선택지
| 모델 | 가격 | 속도 제한 | 특징 |
|------|------|----------|------|
| `grok-imagine-image` | **$0.02/장** | 300 rpm | 기본 모델, 비용 효율 최고 |
| `grok-imagine-image-pro` | **$0.07/장** | 30 rpm | 고품질, 프로급 |

### Data Sharing 이벤트 (핵심!)
- $5 충전 + 학습 동의 → **매달 $150 크레딧 무료**
- 기본 모델 기준: 월 **7,500장** 무료
- Pro 모델 기준: 월 **2,140장** 무료
- console.x.ai → Billing → Data Sharing 토글 ON
