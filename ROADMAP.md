# 🗺️ OurColoring 프로젝트 로드맵

## 프로젝트 개요
- **서비스명:** OurColoring (ourcoloring.com)
- **한줄 설명:** 내 아이 사진을 업로드하면 색칠공부 도안으로 변환해주는 서비스
- **프레임워크:** Astro + React Islands
- **배포:** Cloudflare Pages
- **서버 비용:** $0 (도메인 비용만)

---

## Phase 1: MVP (1주 목표)

### Sprint 1 (Day 1~2): 프로젝트 셋업
- [ ] Astro 프로젝트 초기화 (React 인테그레이션)
- [ ] Cloudflare Pages 배포 파이프라인 구축
- [ ] 프로젝트 구조 확립
- [ ] 다국어(i18n) 기반 구조 세팅 (한국어 우선, 영어)

### Sprint 2 (Day 3~4): 핵심 기능
- [ ] 이미지 업로드 컴포넌트 (드래그앤드롭 + 모바일 카메라)

- [ ] 변환 결과 미리보기
- [ ] 선 굵기/감도 조절 슬라이더

### Sprint 3 (Day 5~6): 다운로드 + UX
- [ ] PNG 다운로드
- [ ] PDF 다운로드 (A4 프린트 최적화)

- [ ] 프린트 가이드 페이지 (와이파이 프린트, 편의점 프린트 안내)

### Sprint 4 (Day 7): 랜딩 + 런칭
- [ ] 랜딩 페이지 (히어로, 사용법, 샘플 결과, CTA)

- [ ] Google Analytics (GA4) 연동
- [ ] Microsoft Clarity 연동 (히트맵/세션 리플레이)
- [ ] AdSense 신청 준비
